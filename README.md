# 项目名称：函数图像计算器

项目目标：提供一个多功能的Python工具，用于绘制数学函数图像、计算交点，并允许用户动态调整坐标系范围。

开发环境：Python 3.8+，Tkinter，Matplotlib，NumPy，SymPy

1. 函数输入与管理：

• 用户可以输入数学函数表达式。

• 提供添加、删除函数的功能。

• 显示已添加的函数列表。


2. 图像绘制：

• 绘制用户输入的函数图像。

• 支持动态调整X轴和Y轴范围。

• 标记并计算函数交点。


3. 用户界面：

• 使用Tkinter创建图形用户界面。

• 提供直观的输入框、按钮和绘图区域。

• 显示使用说明。


4. 交点计算：

• 使用SymPy计算函数交点。

• 在图像上标记交点并显示坐标。
